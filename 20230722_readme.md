리액트는 사용자정의태그를 만드는 기술인다.
사용자정의 태그를 컴포넌트라고 부른다.


props는 입력값이다.
props예시 
```javascript
title="불면증 카운터" initValue="10"
```

{} js의 데이터타입으로 인식
"" 문자열로 인식
함수의 리턴값에서 {}를 쓰면 변수로 간주
